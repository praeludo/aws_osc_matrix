# AWS OSC MATRIX

## Getting started

Retrieve the list of API calls from Cloud providers to compare API compatibility in an xlsx document.

### Prerequisites

* Python 3.5
* requests
* PDFminer
* BeautifulSoup
* Openpyxl
* An .xslx (Microsoft Excel) reader

### Installation

```
pip install -r requierements.txt
```

### Usage
python aws_osc_matrix
